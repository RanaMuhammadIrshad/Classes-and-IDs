Ordered and unordered list

Ordered lists are used when things need to happen in a specific order

  1. wake up
  2. have breakfast
  3. get dressed
  4. start coding

Unordered lists aer ideal when the order is not important (eg. grocery list

  • water
  • bread
  • mango
